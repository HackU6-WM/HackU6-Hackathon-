"DOMAINION" is a domain name suggestion engine built by Team Interpered Language Apologists for Dominion Enterprises'
Hack U 6 Hackathon. The web app was built in a 48 hour period by a team from The College of William and Mary.
"DOMAINION" is built in Python with a Django framework and takes search queries and selected top level domain
lists and returns a list of relevant available domain names to the user.

Relevent APIs Used:
Geonames
Google Maps
Datamuse
NameStudio by Verisign
